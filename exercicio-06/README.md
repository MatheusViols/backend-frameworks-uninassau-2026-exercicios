# Exercício 06: Models e Persistência em JSON

## Objetivo
Criar Model com persistência em arquivo JSON.

## Requisitos
- Model User com CRUD
- Métodos: findAll(), findById(), create(), update(), delete()
- Persistir em `src/data/users.json`
- Sincronizar arquivo a cada operação

## Estrutura
```
src/
├── models/
│   └── User.js
├── data/
│   └── users.json (criado automaticamente)
└── server.js
```

## Comandos
```bash
npm install
npm test
```
