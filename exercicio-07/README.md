# Exercício 07: Views com EJS

## Objetivo
Criar interface web completa com EJS templates.

## Requisitos
- Listar produtos em tabela (index.ejs)
- Formulário para criar produto (create.ejs)
- Partials: header.ejs e footer.ejs
- Rotas: GET /, GET /create, POST /products

## Comandos
```bash
npm install
npm start  # Acesse http://localhost:3000
```
