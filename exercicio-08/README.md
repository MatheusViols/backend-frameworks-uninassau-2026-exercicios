# Exercício 08: CRUD Completo - API REST Book

## Objetivo
Criar API REST completa para entidade Book com validações e tratamento de erros.

## Requisitos
- Model Book: { id, title, author, year, isbn }
- Rotas CRUD completas
- Validações em todas operações
- Tratamento de erros
- Persistência em JSON

## Endpoints
- GET /books - Listar todos
- GET /books/:id - Buscar por ID
- POST /books - Criar
- PUT /books/:id - Atualizar
- DELETE /books/:id - Deletar

## Comandos
```bash
npm install
npm test
```
